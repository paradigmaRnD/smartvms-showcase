# Smart VMS — Government Assessment Showcase

**Paradigma International Limited** · [Government Systems Prototype Showcase](https://gdr.ict.go.ug/innovators-showcase/apply)

This repository is a **read-only assessment package** for the Ministry of ICT and National Guidance (MoICT&NG) and the National Innovator Registry. It is **not** the production codebase.

## Purpose

- Provide architecture and system description documents for evaluation
- Summarise capabilities without exposing proprietary implementation
- Support Registry profiling for future Government digitisation assignments

## Production system

Smart VMS replaces physical visitor logbooks with a secure, multi-tenant digital platform for government and institutional premises (mobile check-in, web administration, audit trails, role-based access).

**Live demonstration:** Contact the innovator via details in submission documents.
